Для использования скрипта - надо уставить веб драйвер для selenium с такой же версией  как и с вашего
браузера chrome. После запуска main.py - скрипт попросит ввести url нужной вам главы манги на сайте
mangalib с первой страницы главы, после этого начнётся загрузка с запуском 2 новых окон, а при успешном
завершении загрузки эти окна закрываются.

create_dir.py - создаёт название директории определённой главы от url, куда будут загружатся страницы.

create_name_for_page.py - создаёт общее название страниц(main.py дополняет названия номером страниц).

input_url.py - для введения url создал отдельный файл, что бы использовать его в остальных файлах.

main.py - тут код для  парсинга сайта и загрузки страниц главы манги в папку.
